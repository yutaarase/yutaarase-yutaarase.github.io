Graw up Shooter

ゲームの概要

  • ジャンルとしては、2D縦画面シューティングゲームです。
  • 敵を倒しお金を稼ぐことができます。
  • お金を消費し自信を強化することが出来ます。

担当

  • 企画、設計、開発

開発要件

  • 使用言語 : Java
  • 統合開発環境 : Eclipse
  • 開発期間 : 2ヶ月
  • 制作期間 : 1ヶ月 2週間

コーディングについて

  • 個人作成のゲームで企画、設計、計画、開発を一人で行っています。
  • ステータスを強化出来るショップを実装しています。
  • レベル制を導入し、レベルごとに弾幕を増やす処理を実装しています。
  • 各ステージにBossを実装し、独自の攻撃を行う様にしています。
